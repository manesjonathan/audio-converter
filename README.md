# Audio Converter
A simple Python audio converter for converting audio files to mp3 format.

## Installation
```bash
$ git clone https://github.com/manesjonathan/audio-converter.git
$ cd audio-converter
$ pip install -r requirements.txt
```

## Usage
```bash
$ python main.py
```
